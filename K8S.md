# KUBERNETES
## Анатомия K8s
__POD__ - минимальная единица K8S, хранит контейнеры, общие тома, общий ip
рекомендуется использовать 1 контейнер в 1 поде.

__Кластер K8S__ хранит в себя узлы (NODE) - которые уже хранят POD
